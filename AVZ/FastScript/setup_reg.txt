    Installing full version of FastScript after the trial install.

ATTENTION! Follow these instructions if you want to install registered files 
over the trial version of FastScript. 

Delphi 4.0

1. Copy fs_iinterpreter.pas and fs_iilparser.pas to FastScript\Source folder.
2. Run Delphi 4.
3. Select menu item Component->Install Packages. Choose "FastScript 1.0" and press 
"Remove" button. Close the window.
4. Select menu item File->Open Project. Open the file fs4.dpk from 
FastScript\Source folder.
5. Press "Compile" button.
6. Select menu item File->Open Project. Open the file dclfs4.dpk from 
FastScript\Source folder.
7. Press "Compile" button and then "Install" button.
8. Close Delphi 4. Choose "No" when Delphi asks to save changes.
9. Copy fs4.bpl file from FastScript\Source to Windows\System32 (or
Windows\System for Windows 9x/Me).
