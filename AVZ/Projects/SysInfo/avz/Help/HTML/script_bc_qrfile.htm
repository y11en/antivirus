function  BC_QrFile

 Наверх  Предыдущая  Последующая |

function  BC_QrFile(AFileName : string) : boolean;

 

Добавляет команду карантина указанного файла в сценарий BootCleaner. В имени файлы допустимы макросы. Карантин файлов посредством BootCleaner полезен в случае, если вредоносная программа блокирует доступ к своим файлам при помощи руткит-механихмов, которые не удается нейтрализовать встроенным антируткитом AVZ. Помещенные в карантин файлы доступны через строенные средства просмотра карантина AVZ, но файлы именуются как bcqrXXXXX.dat и bcqrXXXXX.ini

 

На заметку:

Команда карантина файла имеет приоритет над прочими командами.

 

begin

 // Добавление в сценарий команды карантина файла

 BC_QrFile('%WinDir%\trojan.exe');

 // Настройка протокола

 BC_LogFile(GetAVZDirectory + 'boot_clr.log');

 // Активация

 BC_Activate;

end.
