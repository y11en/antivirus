function InputBox(ACaption, APrompt, ADefault: string): string;

 

Функция InputBox выводит на экран диалоговое окно с полем ввода. Заголовок окна определяется параметром ACaption, приглашение для ввода данных - APrompt. Значение ADefault вводится в строку в качестве значения по умолчанию. В окне имеется две кнопки - OK и Отмена. Работа скрипта останавливается до нажатия на кнопку "ОК" или "Отмена" в окне ввода. При нажатии ОК функция возвращает значение, введенное пользователем. При нажатии кнопки "Отмена" возвращается значение по умолчанию, заданное параметром ADefault.

 

 

var

txt : string;

begin

txt := InputBox('Заголовок окна', 'Введите строку:', 'Строка по умолчанию');

ShowMessage('Введен текст ="'+txt+'"');

end.
