function RegKeyIntParamRead(ARoot, AName, AParam : string) : dword;

 

Производит чтение параметра типа REG_DWORD с имененем AParam ключа AName раздела ARoot.  Указание несуществующего параметра, ключа или раздела не является ошибкой - в этом случае функция возвращает 0.
