Менеджер файла Hosts

 Наверх  Предыдущая  Последующая |

Менеджер файла Hosts позволяет просмотреть содержимое файла Hosts с возможностью удаления любой из его строк. Кроме удаления строк файла Hosts предусмотрена кнопка очистки данного файла, которая оставляет в файле только одну строку "127.0.0.1 localhost", что полезно для оперативной чистки данного файла после его повреждения троянскими программами.

Анализатор менеджера файла Hosts подключен к исследованию системы.
