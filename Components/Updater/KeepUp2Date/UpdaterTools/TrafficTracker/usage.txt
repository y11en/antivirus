TrafficTracker.exe


I) этот инструмент позволяет перехватывать трафик от Brower-a
 и записывать его в лог файлы для дальнейшего анализа.

  I.a) Инструкция использования (через прокси сервер):

   1. для использования трассировки логов необходимо
    настроить Browser: HTTP Proxy Server 127.0.0.1:2000
    и убрать галочку "Bypass proxy server for local addresses"
   2. запустить TrafficTracker.exe с параметром,
    который укажет адрес Вашего Proxy Server-a.
   3. запросить требуемый ресурс в Вашем броузере. Например,
    http://ru3h.kaspersky-labs.com/index/master.xml


  I.b) Инструкция использования (напрямую, без прокси сервера):

   1. для использования трассировки логов необходимо
    настроить Browser *без* использования прокси сервера
   2. запустить TrafficTracker.exe с параметром,
    который укажет адрес конечного ресурса
   3. запросить требуемый Browser-ом адрес:
    http://localhost:2000/index/master.xml



II) также инструмент позволяет записать сессию компонента
  обновления Лаборатории касперского.

  II.a) Инструкция использования (через прокси сервер):

    1) настроить Антивирус использовать прокси 127.0.0.1:2000
    2) запустить TrafficTracker.exe с параметром, который укажет
      адрес Вашего Proxy Server-a или конечного ресурса
    3) произвести обновление


  II.a) Инструкция использования (напрямую, без прокси сервера):

    1) настроить Антивирус *не* использовать прокси сервер
    2) запустить TrafficTracker.exe с параметром,
      который укажет адрес источника обновления
    3) произвести обновление со следующего
      пользовательского ресурса
     127.0.0.1:80



Ш) Результат работы инструмента направить в службу поддержки: файлы
 трассировки сессии, которые образуются рядом с выполяемым файлом


IV) Также в службу тех.поддержки направить следующие данные:
  a) результаты запуска утилиты из командной строки:
    >ipconfig /all
  b) результаты работы утилиты из командной строки:
    >tracert ru3h.kaspersky-labs.com


V) Для получения дополнительной документации, информации
 о параметрах и примеры использования утилиты запустите
    > TrafficTracker.exe -?
