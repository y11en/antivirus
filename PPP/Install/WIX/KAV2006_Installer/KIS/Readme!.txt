Для добавления файла в дистрибутив:

1. Создать новый GUID для файла и описать его в GUIDS.WXI
2. Описать файл в исходнике COMPONENTS.WXS
3. Добавить ссылку на компонент в соответствующую фитчу FEATURES.WXS (некоторые часто используемые группы фитч вынесены в отдельные _*.WXI файлы)
