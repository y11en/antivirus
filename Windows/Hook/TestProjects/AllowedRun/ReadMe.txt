========================================================================
       AllowedRun
========================================================================

Программа предназначена для блокирование доступа и предотвращения запуска из указанных пользователем каталогов
Пользователь может только разрешить каким приложениям (по имени) разрешено открытие (для чтения/записи) в этих каталогах.


Использование:

1. Настройки программы хранятся в файле AllowedRun.ini в каталоге операционной системы (%SystemFolder%).
2. Настройки сохраняются при выходе из программы. Изменения передаются в перехватчик сразу при вводе пользователем.
3. При запуске программы с ключем /autostart программа считывает данные из ini файла, передает настройки в драйвер и 
	завершает свою работу.
4. При запуске без параметров программа считывает текущие настройки из драйвера и позволяет их модифицировать (если до этого программа 
	не была запущена с ключем /autostart то текущие настройки будут заменены на новые)
